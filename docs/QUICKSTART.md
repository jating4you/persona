# persona — Quickstart

## Run locally
From the repo root:

```bash
python3 -m http.server 8080
```

Open:
- http://localhost:8080/ (Landing)
- http://localhost:8080/jatin/job/
- http://localhost:8080/nitin/marriage/
- http://localhost:8080/business/

## Deploy to GitHub Pages (simple)
1. Push this repo to GitHub (repo name: `persona`)
2. Repo → Settings → Pages
3. Source: Deploy from branch
4. Branch: `main`, folder: `/ (root)`
